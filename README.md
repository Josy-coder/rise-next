# RiseNext - Nonprofit Website

A modern nonprofit website with CMS and CRM capabilities, deployable on Cloudflare.


## Features

- Public-facing nonprofit website with:
    - Home, About, Contact pages
    - Programs and Opportunities listings
    - Blog with rich content
    - Application forms for opportunities
    - Application tracking

- Admin dashboard with:
    - Content management for blog posts, programs, and opportunities
    - Application management (CRM functionality)
    - Media library
    - User management
    - Dynamic form builder
